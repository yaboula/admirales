**Quick Setup**
Add your license2 inside config.json -> "owners"
The database will be installed automatically, you just need to restart the resource after it completes (check server console for confirmation)

Documentation and Guides: https://docs.luxu.gg/
Join discord: https://discord.gg/luxu

Notes:
Results might be cached in some cases and not reflect immediate changes. please allow a few minutes for it to refresh.

### Web Panel

Go to https://luxu.app
Make a team with your transaction id
Once you get your key, Paste it in your server.cfg `set luxu_admin_api_key "API_KEY"`
